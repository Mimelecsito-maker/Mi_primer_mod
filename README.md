# Mi Primer Mod - Minecraft Bedrock

Mi primer mod para Minecraft Bedrock.

## Informacion del Mod

- Nombre: Mi Primer Mod
- Version: 1.0.0
- Plataforma: Minecraft Bedrock
- Descripcion: Mi primer mod para Minecraft Bedrock

## Estructura del Proyecto

Mi_primer_mod/
├── manifest.json
├── texts/
│   ├── es_ES.lang
│   └── en_US.lang
├── textures/
├── models/
└── README.md

## Como Usar

1. Descarga este repositorio
2. Comprime la carpeta en formato .zip
3. Renombra el archivo a .mcpack
4. Abre el archivo con Minecraft Bedrock
5. Selecciona el mundo donde quieres activar el mod

## Estructura de Carpetas

- manifest.json - Archivo de configuracion principal (OBLIGATORIO)
- texts/ - Archivos de idioma (.lang)
- textures/ - Texturas personalizadas
- models/ - Modelos personalizados
